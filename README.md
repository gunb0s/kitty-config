# Kitty Configuration

Personal Kitty terminal configuration with Tokyo Night theme.

## Shortcuts

### Tab Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd+T` | New tab |
| `Cmd+W` | Close tab |
| `Cmd+1-9` | Go to tab 1-9 |
| `Ctrl+Shift+Right` | Next tab |
| `Ctrl+Shift+Left` | Previous tab |
| `Ctrl+Shift+.` | Move tab forward |
| `Ctrl+Shift+,` | Move tab backward |

### Window Splits

| Shortcut | Action |
|----------|--------|
| `Cmd+D` | Vertical split |
| `Cmd+Shift+D` | Horizontal split |
| `Cmd+Shift+W` | Close split |

### Navigate Splits

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+H` | Move left |
| `Cmd+Shift+L` | Move right |
| `Cmd+Shift+K` | Move up |
| `Cmd+Shift+J` | Move down |

### Resize Splits

| Shortcut | Action |
|----------|--------|
| `Cmd+Ctrl+H` | Narrower |
| `Cmd+Ctrl+L` | Wider |
| `Cmd+Ctrl+K` | Taller |
| `Cmd+Ctrl+J` | Shorter |
| `Cmd+Ctrl+0` | Reset size |

### Line Editing

| Shortcut | Action |
|----------|--------|
| `Cmd+Left` | Go to beginning of line |
| `Cmd+Right` | Go to end of line |
| `Cmd+Backspace` | Delete to beginning of line |
| `Option+Left` | Move word left |
| `Option+Right` | Move word right |
| `Option+Backspace` | Delete word |

### Other

| Shortcut | Action |
|----------|--------|
| `Cmd+Ctrl+,` | Reload config |
| `Ctrl+Shift+L` | Cycle layouts |
